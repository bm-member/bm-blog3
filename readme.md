### Installation

- git clone https://github.com/bm-member/bm-blog3.git FOLDER_NAME
- cd FOLDER_NAME
- cp .env.example .env
- composer install
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- php artisan serve
- Done!
